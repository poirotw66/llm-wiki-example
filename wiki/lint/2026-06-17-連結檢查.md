# Lint：連結檢查（2026-06-17）

## Summary

- **根因**：OKF `/concepts/foo.md` 在 `wiki/` 嵌於 repo 時，GitHub／IDE 視為磁碟根路徑 → **全斷鏈**。
- **修正**：示範頁改為 **markdown 相對路徑**（如 `../sources/訂單-api-簡介.md`）。
- 規約已更新：`AGENTS.md`、`docs/okf.md`（撰寫 vs 匯出分流）。

## 修復紀錄

| 項目 | 修正 |
|------|------|
| 全示範 wiki 頁 | `/type/file.md` → 相對路徑 |
| [訂單 API 簡介](../sources/訂單-api-簡介.md) Citations | `../../raw/sources/訂單-api-簡介.md` |

## Pass（修正後）

- `wiki/**/*.md` 相對連結均可解析至實體檔案。
- [Index](../index.md) 條目與檔案一致。
- 無 `[[...]]` 混用。

## 規約摘要

| 情境 | 用 |
|------|-----|
| 日常撰寫（本 repo） | `../sources/foo.md` |
| 獨立 OKF bundle 匯出 | 可轉 `/sources/foo.md` |
| 歸檔 Citations | `../../raw/sources/<slug>.md` |
