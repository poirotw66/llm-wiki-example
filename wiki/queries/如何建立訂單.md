# 如何建立訂單？

> **【範例】** 可重用 Query 頁示範；fork 後請刪除或改寫。

## 問題

如何透過訂單服務 API 建立一筆訂單？

## Short Answer

對 `POST /orders` 送 JSON body（含 `customer_id`、`items`、`currency`、建議帶 `idempotency_key`），成功回 **201** 與 `order_id`。（確定）[訂單 API 簡介](../sources/訂單-api-簡介.md)

## Detailed Answer

1. 請求須帶 `Authorization: Bearer <JWT>`，scope 含 `orders:write`。（確定）[訂單 API 簡介](../sources/訂單-api-簡介.md)
2. `POST https://api.example.internal/v1/orders`，`Content-Type: application/json`。（確定）[訂單 API 簡介](../sources/訂單-api-簡介.md)
3. Body 範例含 `items[].sku`、`quantity`、`unit_price`；回應含 `order_id`、`status`（`PENDING`）、`total_amount`。（確定）[訂單 API 簡介](../sources/訂單-api-簡介.md)
4. 相同 `idempotency_key` 重送不重複建單。（確定）[訂單 API 簡介](../sources/訂單-api-簡介.md)
